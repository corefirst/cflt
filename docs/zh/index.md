## 核心命题

> *话语的认知核心，也是其在任何语言中被普遍优先排列的线性位置。*

由此推导出 **核心优先语言方法 (CFLM)** 的四元素排序规则：

```
[核心动作/结果] → [条件/原因] → [空间/语境] → [时间]
```

这一排序既是连接两种自然语言的最小认知摩擦桥梁，也可作为面向大语言模型的结构化提示协议。
