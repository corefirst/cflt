# 方法论：评估与基准测试 (CFLT-Metrics)

> **版本：** 1.0.0 (内部草案)
> **作者：** CFLT 核心团队
> **组织：** [CFLT.center](https://cflt.center)
> **许可：** [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)

> **目的：** 定义用于衡量 CFLT 在人类第二语言习得和 LLM 性能方面有效性的正式指标和实验协议。

---

## 1. 为什么要进行基准测试？

为了从“理论”走向“标准”，CFLT 必须为其主张提供实证。我们在两个领域衡量成功：**人类认知效率**和**人工智能系统稳定性**。

> **关于本文档中数字的说明：** 下面所有的“基准测试”行均为**待实证验证的预期目标**，而非已测量的结果。它们定义了 CFLT 为证明其理论优势而*需要展示*的内容，并为每项未来研究指定了指标、工具和阈值。在存在行业范围基线的地方（例如 SGLang/vLLM 报告的前缀缓存 TTFT 降低），不应将这些通用数字与 CFLT 特有的收益混淆；CFLT 仅通过稳定提示词前缀而继承了这些优势，其增量贡献仍需通过消融实验验证。我们明确邀请独立学术机构进行验证。

---

## 2. 第一支柱指标：人类认知效率

### 2.1 发音起始潜伏期 (Articulation Onset Latency, AOL)
- **定义：** 语义刺激（如图片）与开始说出第一个 L2 单词之间的时间延迟（以毫秒为单位）。
- **CFLT 主张：** CFLT 通过消除生成时的“线性化决策”来降低 AOL。
- **基准：** 与传统基于语法的学习者相比，AOL 降低 30%。

### 2.2 认知负荷指数 (Cognitive Load Index, CLI)
- **定义：** 在用户说 L2 时，通过次要任务表现（如数字跨度测试）进行测量。
- **CFLT 主张：** 固定的 4 插槽协议减少了“前额叶税”（参见 `neuroscience.md`），为其他任务留出更多工作记忆。
- **基准：** 在 L2 生成期间，次要任务的准确率更高。

### 2.3 EIC 效率比
- **定义：** 基于 Hawkins 的 EIC（参见 `linguistics.md`），即“成分识别域”中识别出的成分与单词的比率。
- **基准：** 每个 CFLT 句子的前 2 个 token 目标值为 1.0 (100%)。

---

## 3. 第二支柱指标：人工智能系统稳定性

### 3.1 意图保留得分 (Intent Preservation Score, IPS)
- **定义：** 衡量输出在多大程度上保留了用户初始核心意图的指标（通过余弦相似度或 LLM 评估计算）。
- **CFLT 主张：** CFLT 在位置 0 的“注意汇 (Attention Sink)”对齐防止了意图漂移。
- **基准：** 复杂 4 插槽请求的 IPS > 0.95。

### 3.2 TTFT (首个 Token 时间) 降低
- **定义：** 通过 KV 缓存前缀可重用性实现的延迟降低。
- **CFLT 主张：** 由于 CFLT 强制执行固定的前缀顺序，它在生产环境中最大限度地提高了缓存命中率。
- **基准：** 重复意图类别的 TTFT 降低 50%–80%。

### 3.3 幻觉率 (Hallucination Rate, HR)
- **定义：** 生成的 token 中与提供的上下文或核心动作相矛盾的百分比。
- **CFLT 主张：** 将核心动作锚定在位置 0 减少了模型的“漂移”。
- **基准：** 与自由格式的自然语言提示相比，幻觉率降低 2 倍。

---

## 4. 实验协议

### 4.1 针对 LLM 的 CFLT “消融实验”
1.  **A 组（控制组）：** 单词顺序随机的自然语言提示。
2.  **B 组（CFLT 组）：** 转换为严格 CFLT 序列的相同语义意图。
3.  **任务：** 长上下文指令遵循。
4.  **指标：** “大海捞针 (Needle-in-a-Haystack)”测试的准确率。

### 4.2 针对人类的“双语缓冲”测试
1.  **参与者：** 来自中心语在后 (SOV) 背景的成年 L2 学习者。
2.  **任务：** 将复杂的 L1 句子快速翻译成 L2。
3.  **变量：** 自由翻译 vs. CFLT 支架式翻译。
4.  **指标：** 每 100 个单词的平均停顿次数。

---

## 5. 总结：从数据到标准

通过编纂 these 指标，CFLT 为研究人员和工程师验证该协议提供了一条清晰的路径。我们邀请独立学术机构验证这些基准，以进一步确立 CFLT 作为人类-人工智能同步逻辑的全球标准。
