# CFLT 实证研究议程

> **状态：** 草案 / 提案
> **目标：** 为验证核心优先语言理论 (CFLT) 在计算和教学领域的各项主张提供一个可验证、可证伪的路线图。

## 1. 引言

作为一套关于认知人体工程学的规范性协议，CFLT 对信息处理效率和语言习得速度提出了若干高风险的主张。为了将 CFLT 从一个理论框架转变为一门基于证据的科学，这些主张必须接受严谨的实证检验。本议程概述了三条实证研究 Track：**计算性能**、**心理语言学验证**以及 **SLA（第二语言习得）成果**。（**术语说明：** 这些是实证议程的**方法学研究 Track**，**与** [`../vision.md`](../vision.md) **中编号的两条战略支柱（Pillar I/II）不同**；战略支柱组织项目的高层愿景。）

### 1.1 跨域单一货币前提

该框架的框定性主张——人类 L2 产出与 LLM 提示工程共享一种可由单一协议处理的共同“线性化成本”——是一个**承重的跨域类比**。我们如其所是地对待它，因为这一类比是下文整个研究议程的承重前提，把它当作不言自明会是智识上不诚实的。

**挑战。** Guest & Martin (2023,《On Logical Inference over Brains, Behaviour, and Artificial Neural Networks》)与更早的批评传统（McCloskey 1991）主张，认知架构与神经网络行为之间的表面类比经常夹带未经证成的机制等同。两个系统在相似的表面操作下产生相似的表面行为，**并不**蕴含其底层机制相同。具体而言：人类的“前额叶税”（Levelt 的 Formulator 阶段瓶颈；Kormos 2006）是一种由 DLPFC / LIFG / ACC 调控的工作记忆与控制现象（`foundations/neuroscience.md` §3），而 LLM 的“Lost in the Middle”曲线（Liu et al. 2023）是一种*行为层面*的长上下文性能模式：当承载答案的信息位于长上下文中部、而非靠近开头或结尾时，任务准确率往往下降。Liu 等人是从行为上测量这一点的，并未把某种 softmax 注意力、因果掩码或位置编码机制分离出来作为其成因；CFLT 关于此类机制支撑该曲线的工作假设是它自己要去检验的，而非他们的结论。无论如何，人类与 LLM 现象在任何深层意义上都不是同一机制。

**Slobin (1996)“为说而思 (Thinking for Speaking)”** 提出了第二个、部分正交的挑战：即便*在*人类认知*内部*，语言中立的概念化器（Levelt 1989）也可能比它看上去的更弱。Slobin 主张语言编码塑造了说话者习惯性标记哪些概念区分——即，前言语信息并不像 Levelt 架构所暗示的那样语言中立。如果 Slobin 正确，CFLT 的“共享概念化器支架”会失去一部分承重力：即便经过 CFLT 训练，L1 塑形的支架也可能无法干净地迁移到 L2 产出。

**CFLT 的回应——联合的、而非单一的货币。**

我们接受 Guest & Martin / Slobin 的批评，并据此修订该框定性主张：

1. **CFLT *不*主张人类产出与 LLM 提示工程之间存在单一共享机制。** 两个系统以*不同的机制货币*付出线性化成本——一者是 DLPFC 代谢负荷，另一者是注意力在位置上的分布。跨域主张不是说机制相同；而是更弱的主张：**单一规范性协议——核心优先线性化——恰好在两个系统中都是有用的干预**，原因部分不同。
2. **“共享货币”是那个*干预*，而非那个*成本*。** CFLT 实际提议的是单一的一件工程——`Core → R → S → T` 协议——并（可检验地）预测它同时降低人类的 DLPFC 重构成本（P3，`core-concept.md` §8.5）*与* LLM 的注意力漂移（P2）。这是两个可独立证伪的独立预测。证伪其一并不证伪其二。
3. **Slobin 的挑战是部分性的。** Slobin 的“为说而思”提示概念化器受 L1 塑形的习惯影响；它并不提示概念化器*完全*由 L1 决定。折中的解读是：CFLT 明确地为线性化阶段搭建支架，这可能*衰减*（而非消除）L1 塑形的概念化器偏差。这是可检验的：一个经 CFLT 训练的汉语-L1 / 英语-L2 说话者，应当比未经训练的匹配对照表现出*更少*的 L1 塑形时间前置（P1，`core-concept.md` §8.5）。

**统一主张的可证伪条件。** 如果 P2（LLM 侧）与 P1（人类侧）各自都被证伪，则“联合货币”主张坍塌——CFLT 要么成为一个无 LLM 收益的人类侧支架，要么成为一个无教学收益的 LLM 侧提示协议，要么两者皆非。本议程的三 Track 结构（§2 计算、§3 心理语言学、§4 SLA）就是**为让这种分离可被观察而设计的**：我们不假装单一实验能验证这一统一；我们在每条 Track 中运行独立实验，并让跨域主张依其联合结果而存续或倒下。

这是一个比宣言中“思想的 TCP/IP”隐喻刻意更弱的框定。该隐喻在修辞上有用；承重的科学主张是各 Track 可检验预测的合取，而单一货币的解读被收回到*战略定位*层。

### 1.2 范围说明 —— 一个单独保留的开放问题

本议程检验的是 Core 的**位置**（Core 前置 vs Core 后置；见 §2.1 的 *Core-Modifier vs Modifier-Core*）。它**尚未**确定**地面框架的内部架构**——*为何*顺序是 Reason → Space → Time、*为何*是这一组槽。该尾巴架构的证成在此作为**开放问题、正在主动研究中**呈现，而非已定论；其研究设计将**预注册**。（Core 位置的主张另有支撑，见 §2；二者不应混为一谈。）

## 2. 计算 Track：计算性能 (LLM 提示工程)

核心优先序列是否能真正提高大语言模型 (LLM) 的推理和准确性？

### 2.1 实验：“指令优先”一致性测试
*   **假设：** 与将指令放在末尾相比，将核心指令置于长上下文的开头（CFLT 顺序）能减少“中间丢失 (middle-loss)”现象，并提高指令遵循的准确性。
*   **自变量：** 指令位置（CFLT：核心-修饰语 vs. 反向：修饰语-核心）。
*   **因变量：** 
    *   **准确率：** 正确执行任务的百分比。
    *   **一致性：** 100 次运行中输出格式的标准差。
    *   **令牌效率：** 达到正确解决方案所需的平均 token 数。
*   **衡量指标：** LLM 作为评委的评分和确定性单元测试。

### 2.2 实验：跨语言推理映射
*   **假设：** 在低资源语言对中，使用 CFLT 作为跨语言推理任务的“枢轴逻辑”能减少翻译导致的逻辑错误。
*   **方法：** 比较 [L1 提示词 → 推理 → L2 回答] 与 [L1 提示词 → CFLT 枢轴 → 推理 → L2 回答]。

## 3. 心理语言学 Track：心理语言学验证 (人类处理)

人脑是否能以更低的代谢/认知成本处理核心优先结构？

### 3.1 实验：自定步速阅读 (SPR) 与反应时
*   **假设：** 与核心被多个情境修饰语先行引导的情况相比，当核心出现在初始位置时，学习者 (L2) 和母语者 (L1) 对“核心”部分的阅读时间会更短。
*   **衡量指标：** 总阅读时间 (TRT) 和通过眼动追踪或 SPR 延迟测得的首次注视时长 (FFD)。

### 3.2 实验：通过双任务范式的认知负荷测量
*   **假设：** 使用 CFLT 协议生成 L2 句子比生成传统结构的 L2 句子消耗更少的工作记忆。
*   **方法：** 受试者执行主任务（口语 L2 产出）和次任务（例如音调监测）。
*   **衡量指标：** 对次任务的反应时间。次任务反应时间越快，表明主任务 (CFLT) 的认知负荷越低。

## 4. SLA Track：SLA 成果 (教学法功效)

CFLT 训练是否能带来更快的流利度和更高的准确性？

### 4.1 纵向 A/B 研究：“流利度飞跃”
*   **假设：** 与使用传统语法优先方法的对照组相比，接受过 CFLT 支架训练的学习者在即兴演说中将实现更高的**平均无停顿字数 (MLR)** 和更少的“填充停顿”（呃/嗯）。
*   **参与者：** 两组 A1 级别的成年学习者（每组 n=50）。
*   **持续时间：** 12 周的教学。
*   **衡量指标：** 
    *   **流利度：** WPM（每分钟字数）、MLR。
    *   **准确性：** 无错误从句百分比。
    *   **复杂度：** AS-unit 复杂度指数。

### 4.2 实验：“僵化”检查 (Skehan 权衡)
*   **目标：** 确定 CFLT 支架的最佳“消退”点。
*   **方法：** 比较继续严格使用 CFLT 的高级学习者 (B2+) 与被鼓励使用“有标记”形式（去支架化）的学习者。
*   **衡量指标：** 母语者对“自然度”和“修辞效果”的主观评分。
