# CFLT 实证研究议程

> **状态：** 草案 / 提案
> **目标：** 为验证核心优先语言理论 (CFLT) 在计算和教学领域的各项主张提供一个可验证、可证伪的路线图。

## 1. 引言

作为一套关于认知人体工程学的规范性协议，CFLT 对信息处理效率和语言习得速度提出了若干高风险的主张。为了将 CFLT 从一个理论框架转变为一门基于证据的科学，这些主张必须接受严谨的实证检验。本议程概述了三个研究支柱：**计算性能**、**心理语言学验证**以及 **SLA（第二语言习得）成果**。

---

## 2. 支柱 I：计算性能 (LLM 提示工程)

核心优先序列是否能真正提高大语言模型 (LLM) 的推理和准确性？

### 2.1 实验：“指令优先”一致性测试
*   **假设：** 与将指令放在末尾相比，将核心指令置于长上下文的开头（CFLT 顺序）能减少“中间丢失 (middle-loss)”现象，并提高指令遵循的准确性。
*   **自变量：** 指令位置（CFLT：核心-修饰语 vs. 反向：修饰语-核心）。
*   **因变量：** 
    *   **准确率：** 正确执行任务的百分比。
    *   **一致性：** 100 次运行中输出格式的标准差。
    *   **令牌效率：** 达到正确解决方案所需的平均 token 数。
*   **衡量指标：** LLM 作为评委的评分和确定性单元测试。

### 2.2 实验：跨语言推理映射
*   **假设：** 在低资源语言对中，使用 CFLT 作为跨语言推理任务的“枢轴逻辑”能减少翻译导致的逻辑错误。
*   **方法：** 比较 [L1 提示词 → 推理 → L2 回答] 与 [L1 提示词 → CFLT 枢轴 → 推理 → L2 回答]。

## 3. 支柱 II：心理语言学验证 (人类处理)

人脑是否能以更低的代谢/认知成本处理核心优先结构？

### 3.1 实验：自定步速阅读 (SPR) 与反应时
*   **假设：** 与核心被多个情境修饰语先行引导的情况相比，当核心出现在初始位置时，学习者 (L2) 和母语者 (L1) 对“核心”部分的阅读时间会更短。
*   **衡量指标：** 总阅读时间 (TRT) 和通过眼动追踪或 SPR 延迟测得的首次注视时长 (FFD)。

### 3.2 实验：通过双任务范式的认知负荷测量
*   **假设：** 使用 CFLT 协议生成 L2 句子比生成传统结构的 L2 句子消耗更少的工作记忆。
*   **方法：** 受试者执行主任务（口语 L2 产出）和次任务（例如音调监测）。
*   **衡量指标：** 对次任务的反应时间。次任务反应时间越快，表明主任务 (CFLT) 的认知负荷越低。

## 4. 支柱 III：SLA 成果 (教学法功效)

CFLT 训练是否能带来更快的流利度和更高的准确性？

### 4.1 纵向 A/B 研究：“流利度飞跃”
*   **假设：** 与使用传统语法优先方法的对照组相比，接受过 CFLT 支架训练的学习者在即兴演说中将实现更高的**平均无停顿字数 (MLR)** 和更少的“填充停顿”（呃/嗯）。
*   **参与者：** 两组 A1 级别的成年学习者（每组 n=50）。
*   **持续时间：** 12 周的教学。
*   **衡量指标：** 
    *   **流利度：** WPM（每分钟字数）、MLR。
    *   **准确性：** 无错误从句百分比。
    *   **复杂度：** AS-unit 复杂度指数。

### 4.2 实验：“僵化”检查 (Skehan 权衡)
*   **目标：** 确定 CFLT 支架的最佳“消退”点。
*   **方法：** 比较继续严格使用 CFLT 的高级学习者 (B2+) 与被鼓励使用“有标记”形式（去支架化）的学习者。
*   **衡量指标：** 母语者对“自然度”和“修辞效果”的主观评分。
